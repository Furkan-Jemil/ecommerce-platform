description: How to run the E-Commerce Platform
---

Follow these steps to get the full platform (Frontend + Backend) running:

### 1. Backend Server (API)
Open a terminal and run:
// turbo
```bash
npm run dev:server
```
*Wait for: `Server is running on http://localhost:3000`*

### 2. Frontend App (React Router)
Open a **second terminal** and run:
// turbo
```bash
npm run dev
```
*Wait for: `âžœ Local: http://localhost:5173/`*

### 3. Usage
- **View Store**: [http://localhost:5173](http://localhost:5173)
- **Check API**: [http://localhost:3000/api/health](http://localhost:3000/api/health)

> [!NOTE]
> Ensure both terminals remain open while using the application.
